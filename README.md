# 🚀 Responsive Startup Landing Page

## 📌 About
This project is a responsive startup landing page built using HTML and CSS.  
It demonstrates modern UI design, layout structuring, and responsive web development techniques.

---

## 🎯 Features
- Clean and modern UI  
- Responsive design for mobile and desktop  
- Flexbox layout for alignment  
- Hero section with call-to-action  
- Feature cards with hover effects  
- Simple and structured layout  

---

## 🛠 Tech Stack
- HTML5  
- CSS3  

---

## 📂 Project Structure

landing-page/
│── index.html  
│── style.css  

---

## 📸 Preview
A modern landing page with a navigation bar, hero section, feature cards, and footer designed to simulate a startup website.
